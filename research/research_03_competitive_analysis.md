## 경쟁 분석: 기회와 위협

### 기회 (Blue Ocean)
- 어린이집 대상 알레르기 알림 서비스는 **국내 공백 시장**
- NEIS API는 학교만 지원 → 어린이집(키즈노트)은 미커버
- 학부모의 알레르기 관련 불안 및 관심 고조
- 어린이집 식단표 알레르기 표기 의무화로 데이터 기반 확보 가능

### 위협
- 키즈노트 공식 API 미제공 → 데이터 수집 불확실성
- 키즈노트가 자체 알레르기 알림 기능 추가 가능
- 개인정보(아동 알레르기 정보) 보호 규정 강화

## 차별화 포인트

1. **어린이집 특화**: 학교 급식 앱과 달리 어린이집/유치원 타깃
2. **개인 알레르기 프로필**: 아이별 알레르기 등록 → 맞춤 알림
3. **전날 알림**: 식단표 업로드 후 해당 메뉴 제공 전날 자동 발송
4. **대체 간식 안내**: 단순 알림이 아닌 대체 간식 준비 안내까지
5. **다채널 알림**: 카카오톡 알림톡 + 푸시 알림 + SMS
