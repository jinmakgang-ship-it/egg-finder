# 리서치 02: 한국 식품 알레르기 번호 체계

## 개요

한국은 **19가지 항목, 총 22가지 식품**에 대해 알레르기 유발물질 표시를 의무화.
(식품의약품안전처 기준, 2018년 4월 잣 추가로 현행 19번까지)

## 알레르기 유발물질 번호-물질 매핑 테이블

| 번호 | 알레르기 유발물질 | 표기 방식 | 비고 |
|-----|----------------|---------|------|
| **1** | **알류(가금류)** | **계란, 달걀, 난류** | **닭·오리·거위 등 가금류만 해당** |
| 2 | 우유 | 우유, 카제인, 유청단백 | 함량 무관 표시 의무 |
| 3 | 메밀 | 메밀 | 국내 중요 알레르기 원인 |
| 4 | 땅콩 | 땅콩 | 견과류 카테고리 |
| 5 | 밀 | 밀, 밀가루 | 주요 알레르기 유발 곡물 |
| 6 | 고등어 | 고등어 | 어류 카테고리 |
| 7 | 게 | 게 | 갑각류 카테고리 |
| 8 | 대두 | 대두, 콩 | 두유, 두부 포함 |
| 9 | 복숭아 | 복숭아 | 과일류 |
| 10 | 토마토 | 토마토 | 과일/채소류 |
| 11 | 새우 | 새우 | 갑각류 카테고리 |
| 12 | 돼지고기 | 돼지고기 | 육류 |
| 13 | 쇠고기 | 쇠고기 | 육류 |
| 14 | 아황산류 | 아황산나트륨 등 | 최종제품 SO₂ ≥10mg/kg 시 표시 |
| 15 | 호두 | 호두 | 견과류 카테고리 |
| 16 | 닭고기 | 닭고기 | 육류 (1번 알류와 별도) |
| 17 | 오징어 | 오징어 | 연체류 |
| 18 | 조개류 | 굴, 전복, 홍합 등 | 연체류/패류 카테고리 |
| 19 | 잣 | 잣 | 2018년 4월 추가 |

> **계란 알레르기 = 1번 (알류)**

## 실제 식단표 표기 예시

```
메뉴명: 계란말이
알레르기 유발물질: 1

메뉴명: 우유 핫케이크
알레르기 유발물질: 1.2.5

메뉴명: 새우튀김 우동
알레르기 유발물질: 5.11
```

키즈노트 식단표에서는 메뉴명 옆에 위와 같이 번호가 병기됨.

## 어린이집 알레르기 표기 법적 의무

**법적 근거**: 보육사업안내 (어린이집 관리지침)

| 의무 항목 | 내용 |
|---------|------|
| 정보 안내 | 알레르기 유발물질 정보를 사전에 보호자에게 통지 |
| 보육일지 기록 | 유병어린이 조사 및 파악 |
| 특별 관리 | 알레르기 아동 별도 식단 준비 |
| 응급 대처 | 아나필락시스 응급처치 계획 수립 |
| 교사 교육 | 보수교육 실시 의무 |
| 식단표 게시 | 월간(가정통신문·홈페이지), 주간(식당·교실) |

## 서비스 구현 시 알레르기 파싱 로직

```
식단표 텍스트 예시: "계란말이 1, 된장국 5.8, 우유 2"

파싱 규칙:
- 숫자 1 검출 → 알류(계란) 포함 메뉴로 분류
- 복수 번호: 마침표(.) 또는 쉼표(,) 구분자
- 메뉴별 알레르기 번호 리스트 추출 후 사용자 알레르기 프로필과 매칭
```
