📖 사용법
1단계 — 텔레그램 봇 만들기
텔레그램 앱 → @BotFather 검색 → /newbot → Token 복사

2단계 — Chat ID 확인
봇에 아무 메시지 전송 후,
api.telegram.org/bot{TOKEN}/getUpdates 접속
→ "id" 값이 Chat ID

3단계 — 식단 등록
키즈노트 식단표 → 스크린샷 → + 탭에서 등록

4단계 — 서버 자동 알림 (권장)
Google Apps Script를 연결하면 폰을 안 켜도 매일 9시에 자동 발송돼요.
설정 탭 → ☁️ 서버 자동 알림 → Web App URL 입력
