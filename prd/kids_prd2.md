## 1. 개요

### 한 줄 정의
키즈노트 식단표를 **모바일로 사진 찍어 올리면**, 네이버 CLOVA OCR로 텍스트를 추출하고 계란이 포함된 메뉴를 자동으로 찾아 **전날 저녁 Telegram 무료 알림**을 보내주는 가족 전용 도구.

### 해결하는 문제
- 키즈노트에 식단표가 올라오지만 매일 확인하지 않으면 계란 포함 메뉴를 놓치게 됨
- 계란 알레르기가 있는 아이를 위해 대체 간식을 챙겨야 하는데, 당일 아침에 알아채면 이미 늦음
- **전날 저녁에 알려줘야** 다음 날 등원 전에 대체 간식을 준비할 수 있음

### 사용 대상
- 우리 가족 (계란 알레르기 아이 1명)
- 별도 회원가입, 프로필 등록 불필요

## 2. 핵심 기능

### 2.1 식단표 이미지 업로드 (모바일 카메라)

모바일 웹 브라우저에서 키즈노트 식단표 화면을 캡처하거나 사진 찍어 업로드.

**사용 흐름**
1. 키즈노트 앱에서 식단표 화면을 스크린샷 또는 사진 촬영
2. 모바일 브라우저에서 서비스 접속
3. 이미지 업로드 버튼 탭 → 갤러리 또는 카메라 선택
4. 날짜 선택 후 [분석하기] 탭

**모바일 이미지 업로드 UI**
```
┌───────────────────────┐
│  🥗 계란 알림이        │
│                        │
│  날짜: [3월 23일 ▼]   │
│                        │
│  ┌────────────────┐   │
│  │                │   │
│  │   📷 사진 찍기  │   │
│  │   🖼️ 갤러리    │   │
│  │                │   │
│  └────────────────┘   │
│                        │
│     [분석하기]         │
└───────────────────────┘
```

- `<input type="file" accept="image/*" capture="environment">` — 모바일 카메라 직접 호출 지원
- 일주일치 식단표 이미지를 한 장에 찍어 올리면 날짜별로 파싱
- 저장된 식단은 캘린더 뷰로 확인

### 2.2 네이버 CLOVA OCR 텍스트 추출

업로드된 이미지를 네이버 CLOVA OCR API로 텍스트 변환.

**CLOVA OCR 선택 이유**

| 항목 | CLOVA OCR | Google Vision | Tesseract |
|------|-----------|--------------|-----------|
| 한국어 인식률 | **매우 높음** | 높음 | 보통 |
| 무료 한도 | **월 100건** | 월 1,000건 | 완전 무료 |
| 모바일 호환 | O | O | 서버 설치 필요 |
| API 난이도 | 쉬움 | 보통 | 복잡 |

> 식단표 등록은 주 1~2회 수준 → 월 100건 무료 한도 충분

**CLOVA OCR API 호출 방식**
```
POST https://invoke.apigw.ntruss.com/custom/v1/{API_URL}
Headers:
  X-OCR-SECRET: {SECRET_KEY}
  Content-Type: application/json

Body:
{
  "version": "V2",
  "requestId": "uuid",
  "timestamp": 0,
  "images": [{
    "format": "jpg",
    "name": "meal_plan",
    "data": "{base64_encoded_image}"
  }]
}
```

**실제 식단표 형식 (확인됨)**

키즈노트 식단표는 월간 표 형태로, 각 날짜 칸 구성:
```
[날짜]
오전간식: 메뉴명
점심: 메뉴명
오후간식: 메뉴명
알레르기/식품: 4,5,7/1,9
```

알레르기 번호는 날짜별로 **"알레르기/식품:" 행에 쉼표로 묶여** 표시됨.
슬래시(`/`) 앞은 알레르기 번호, 뒤는 식품 첨가물 번호로 추정.

**OCR 결과 → 식단 파싱 흐름**
```
[월간 식단표 이미지]
  ↓ CLOVA OCR (전체 텍스트 추출)
[추출 텍스트 예시]
  "3일(월)
   오전간식: 대체공휴일
   점심: ...
   알레르기/식품: 4,5,7/1,9

   4일(화)
   오전간식: 딸기
   점심: 이유식감자수프 ...
   알레르기/식품: 4,1,5/1,9"
  ↓ 파싱 엔진
[날짜별 알레르기 번호 추출]
  3일: [4,5,7] → 1번 없음 → 계란 없음
  4일: [4,1,5] → 1번 있음 → 계란 포함!
```

**파싱 핵심 로직**
- `알레르기/식품:` 패턴 뒤 숫자 추출
- 슬래시(`/`) 기준 앞부분(알레르기 번호)에서 `1` 존재 여부 확인
- 보조: 메뉴명에 `계란`, `달걀` 키워드 직접 포함 여부도 병행 체크

---

### 2.3 계란 포함 메뉴 자동 감지

**파싱 규칙**
- 식단 텍스트에서 숫자 `1` (알류/계란) 포함 여부 확인
- 구분자: `.` (점) 또는 `,` (쉼표)
- 감지 키워드도 병행: `계란`, `달걀`, `난`, `에그`

**파싱 예시**
```
입력: "계란샌드위치 1.5, 우유 2"
결과: 계란샌드위치 → 계란 포함 ✓ (번호 1 감지)

입력: "잡곡밥, 된장국 5.8"
결과: 계란 없음

입력: "계란찜 1"
결과: 계란찜 → 계란 포함 ✓
```

**감지 결과 화면**
```
📅 3월 23일 (월) 계란 포함 메뉴

⚠️ 오전간식: 계란샌드위치
⚠️ 점심: 계란찜

→ 내일 대체 간식을 챙겨주세요!
```

### 2.4 전날 저녁 알림 발송

계란 포함 메뉴가 있는 날의 **전날 저녁 설정 시간**에 자동 알림 발송.

**알림 채널: Telegram Bot (무료)**

Telegram Bot이 가족 채팅방에 메시지를 보내는 방식.

**알림 메시지 예시**
```
🥚 내일 어린이집 계란 메뉴 알림

📅 내일: 3월 23일 (월)

⚠️ 계란 포함 메뉴
- 오전간식: 계란샌드위치
- 점심: 계란찜

👜 내일 등원 시 대체 간식을 챙겨주세요!
```

**알림이 없는 날**
```
✅ 내일(3월 23일) 식단에 계란 메뉴 없어요. 안심하세요!
```

**알림 설정**
- 기본 발송 시간: 오후 8시
- 계란 메뉴가 없는 날은 선택적으로 알림 끄기 가능

## 3. 알림 방식 상세: Telegram Bot

### Telegram Bot을 선택한 이유

| 항목 | Telegram Bot | 카카오 알림톡 | 이메일 |
|------|------------|------------|------|
| 비용 | **완전 무료** | 건당 ~8원 | 무료 |
| 설정 난이도 | 쉬움 | 복잡 (사업자 필요) | 보통 |
| 수신 편의성 | 앱 푸시 알림 | 카카오톡 알림 | 이메일 앱 |
| 가족 공유 | 그룹 채팅방 가능 | 개인만 | CC 가능 |
| API 제공 | **공식 무료 API** | 유료 | SMTP |

### 설정 방법 (최초 1회)

1. Telegram 앱에서 `@BotFather` 검색
2. `/newbot` 명령으로 봇 생성 → **Bot Token** 발급
3. 가족 Telegram 채팅방 또는 개인 채팅방에 봇 초대
4. 채팅방의 **Chat ID** 확인
5. 서비스 환경변수에 Token + Chat ID 입력 → 완료

### API 호출 방식

```
Telegram Bot API (무료, 공식)

POST https://api.telegram.org/bot{TOKEN}/sendMessage
{
  "chat_id": "{CHAT_ID}",
  "text": "🥚 내일 어린이집 계란 메뉴 알림\n...",
  "parse_mode": "HTML"
}
```

## 7. 계란 감지 로직 상세

```python
import re

EGG_KEYWORDS = ["계란", "달걀", "난류", "알류"]

def has_egg_in_allergen_row(allergen_text: str) -> bool:
    """
    "알레르기/식품: 4,1,5/1,9" 형식에서 계란(1번) 포함 여부 확인
    슬래시 앞부분(알레르기 번호)에서 1 존재 여부 체크
    """
    # "알레르기/식품:" 이후 텍스트 추출
    match = re.search(r'알레르기[/식품]*[:\s]+([\d,/]+)', allergen_text)
    if match:
        allergen_part = match.group(1).split('/')[0]  # 슬래시 앞부분만
        nums = [n.strip() for n in allergen_part.split(',')]
        if '1' in nums:
            return True

    # 보조: 메뉴명에 키워드 직접 포함 여부
    for keyword in EGG_KEYWORDS:
        if keyword in allergen_text:
            return True

    return False

def parse_monthly_meal_plan(ocr_text: str) -> dict:
    """
    OCR 추출 텍스트에서 날짜별 계란 포함 여부 반환
    반환: {"2026-03-04": True, "2026-03-05": False, ...}
    """
    results = {}
    # 날짜 패턴: "4일(화)", "10일(월)" 등
    day_blocks = re.split(r'(\d{1,2}일\([월화수목금]\))', ocr_text)

    current_day = None
    for block in day_blocks:
        day_match = re.match(r'(\d{1,2})일\([월화수목금]\)', block)
        if day_match:
            current_day = int(day_match.group(1))
        elif current_day and '알레르기' in block:
            results[current_day] = has_egg_in_allergen_row(block)

    return results
```

## 8. 리스크

| 리스크 | 대응 방안 |
|-------|---------|
| OCR 인식 오류 (흐린 사진, 손글씨) | 결과 화면에서 수동 수정 UI 제공 |
| 알레르기 번호 없이 메뉴명만 있는 경우 | 계란/달걀 키워드 감지로 보완 |
| CLOVA OCR 월 100건 초과 | 가족 전용이라 초과 가능성 낮음 (주 1~2회 = 월 8건 수준) |
| 식단 미입력 날 알림 누락 | 미입력 날은 "식단 미등록" 알림으로 직접 확인 유도 |
| Render 무료 티어 슬립 모드 | APScheduler 알림은 서버가 깨어있어야 함 → UptimeRobot 무료 핑으로 슬립 방지 |
