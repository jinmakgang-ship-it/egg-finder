## 1. 알림 시간 변경

### 문제
저녁 8시 알림은 다음 날 아침 준비가 촉박함. 대체 간식을 마트에서 구입하거나 준비할 시간이 없음.

### 변경
- **전날 오전 9시** 알림으로 변경
- 하루 전 낮에 알림을 받아 여유 있게 준비 가능
- 설정 탭에서 시간 변경 가능 (기본값: 09:00)

### 알림 타이밍 예시
```
수요일 식단에 계란 포함
  → 화요일 오전 9시에 텔레그램 알림 발송
  → 화요일 낮/저녁에 대체 간식 준비
  → 수요일 등원 시 간식 지참
```

## 2. 대체 음식 추천 기능

### 개요
계란 포함 메뉴를 파악한 후, 해당 메뉴와 **같은 계열의 대체 음식**을 추천.
별도 API 불필요 — 메뉴명 키워드 + 식사 시간대(슬롯) 기반 로직으로 추천.

### 추천 로직

**Step 1 — 메뉴명 키워드 매칭** (우선)
계란 메뉴의 이름에서 요리 유형을 파악하여 같은 계열 대체 음식 추천.

| 계란 메뉴 키워드 | 카테고리 | 추천 대체 음식 |
|--------------|--------|-------------|
| 계란찜, 달걀찜 | 찜 반찬 | 두부찜, 순두부, 감자조림 |
| 계란말이, 달걀말이 | 볶음/말이 반찬 | 어묵볶음, 두부구이, 멸치볶음 |
| 계란국, 달걀국, 계란탕 | 국 | 미역국, 콩나물국, 두부된장국 |
| 계란후라이 | 구이 반찬 | 두부부침, 고구마조림, 콩자반 |
| 계란샌드위치 | 빵 간식 | 치즈샌드위치, 잼토스트, 크래커+치즈 |
| 스크램블, 오믈렛 | 볶음 반찬 | 감자볶음, 야채볶음, 두부스크램블 |
| 계란볶음밥, 달걀볶음밥 | 밥 | 참치볶음밥, 야채볶음밥, 주먹밥 |
| 케이크, 머핀, 쿠키, 빵 | 구운 간식 | 쌀과자, 고구마, 과일, 요거트 |

**Step 2 — 슬롯(식사 시간대) 기반 폴백** (키워드 매칭 실패 시)
메뉴명으로 유형 파악이 안 될 때 식사 시간대에 따라 기본 추천.

| 시간대 | 추천 대체 음식 |
|-------|-------------|
| 오전간식 | 과일(바나나/사과/귤), 요거트, 치즈+크래커, 고구마 |
| 오후간식 | 과일(바나나/사과/귤), 요거트, 치즈+크래커, 쌀과자 |
| 점심 | 두부조림, 멸치볶음, 콩나물무침, 시금치나물 |
| 아침 / 기타 | 과일, 치즈, 두부, 삶은 고구마 |

### 텔레그램 알림 메시지 예시

```
🥚 내일 어린이집 계란 메뉴 알림

📅 내일: 3월 23일 (월)

⚠️ 계란 포함 메뉴
• 오전간식: 계란샌드위치
• 점심: 계란찜

🍱 대체 음식 추천
[오전간식 대신]
→ 치즈샌드위치, 잼토스트, 크래커+치즈

[점심 대신]
→ 두부찜, 순두부, 감자조림

👜 내일 등원 시 대체 간식을 챙겨주세요!
```

## 3. 필요한 API 키

### Telegram Bot (무료, 필수)

| 항목 | 설명 | 발급 방법 |
|------|------|---------|
| Bot Token | 봇 인증 토큰 | 텔레그램 앱 → @BotFather → `/newbot` |
| Chat ID | 메시지 수신 채팅방 ID | 봇에 메시지 전송 후 `api.telegram.org/bot{TOKEN}/getUpdates` 접속 → `"id"` 값 |

**발급 단계**
```
1. 텔레그램 앱에서 @BotFather 검색
2. /newbot 입력 → 봇 이름 설정
3. Bot Token 발급 (예: 123456789:ABCDEFxyz...)
4. 본인 또는 가족 채팅방에 봇 초대
5. 봇에게 아무 메시지 전송
6. 브라우저에서 아래 주소 접속:
   https://api.telegram.org/bot{TOKEN}/getUpdates
7. "id": 숫자 → 이게 Chat ID
```

---

### Naver CLOVA OCR (월 100건 무료, 선택)

| 항목 | 설명 | 발급 방법 |
|------|------|---------|
| API Gateway URL | OCR 호출 엔드포인트 | NCP 콘솔 → AI·NAVER API → CLOVA OCR → 도메인 생성 |
| Secret Key | 인증 키 | 도메인 생성 후 자동 발급 |

**발급 단계**
```
1. https://www.ncloud.com 가입 (카카오/네이버 소셜 가능)
2. 콘솔 → AI·NAVER API → CLOVA OCR
3. [도메인 생성] 클릭
4. 서비스 유형: 일반 문서
5. 생성 완료 후 → API Gateway URL + Secret Key 복사
6. 앱 설정 탭에 입력
```

> OCR 없이도 텍스트 직접 입력으로 사용 가능.
